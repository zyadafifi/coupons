## 📋 Maintenance Going Forward

### Before Committing
Always check you're not committing:
```bash
# Check git status
git status

# Look for these (should be ignored):
# - dist/
# - node_modules/
# - android/.gradle/
# - *.env (except env.example)
# - *.zip, *.tar.gz
```

### Regular Cleanup Commands
```bash
# Clean build artifacts
npm run build:web  # Rebuilds dist/
rm -rf dist/       # Remove build output

# Clean Android builds
cd android
./gradlew clean
cd ..

# Clean node_modules (if needed)
rm -rf node_modules
npm install
```

### Before Deploying
```bash
# Verify builds work
npm run build:web         # For Netlify
npm run build:mobile      # For mobile apps

# Verify no secrets committed
git diff                  # Check staged changes
git log --oneline -5      # Check recent commits
```
