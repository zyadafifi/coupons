Q: Splash still looks old/white?
A: Uninstall app completely first!
   adb uninstall com.coupons.app

Q: Logo too small?
A: Edit scripts/generate-splash.mjs:
   logoWidth: 1400 (increase from 1200)
   Then: npm run splash:android

Q: Splash not updating?
A: Clean build:
   cd android && ./gradlew clean && cd ..
   npm run build:mobile
